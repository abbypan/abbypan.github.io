#  Debian : 安装bind9，配置允许本地递归查询

``apt-get install bind9 dnsutils``

编辑``/etc/bind/named.conf.options``，在
{% highlight bash %}
options{
    ....
};
{% endhighlight %}
内添加
{% highlight bash %}
allow-recursion {
    127.0.0.1;
}; 
{% endhighlight %}

如果提示 rndc: connect failed: 127.0.0.1#953: connection refused  

``rndc-confgen > /etc/bind/rndc.conf``

测试bind

``named -g``

重启bind

``/etc/init.d/bind9 restart``
