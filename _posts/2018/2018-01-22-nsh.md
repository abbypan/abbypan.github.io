# 笔记

字段细节回头抽空再看

试图统一数据包在专用网内的策略流转接口

控制面下发策略到 Service Function Forwarder

数据包在Service Classifier入口处，就通过添加NSH头部，逐跳按预设好的策略在Service Function Forwarder链式流转

所以路径是相对固定可控的，host to end，例如：

    NAT -> FW -> DPI -> ... -> LB

可以与vxlan结合使用

这个路由器升级支持是不是又可以卖一波钱啊……
